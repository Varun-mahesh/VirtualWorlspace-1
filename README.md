Quick Notes App
A small Flask app that lets you save and delete notes.

What it does
-Shows a simple notes page
-Saves notes in a local SQLite database
-Lets you view saved notes
-Lets you delete saved notes

Files
~app.py — Flask backend
~index.html — page layout
~script.js — web logic for loading, saving, deleting notes
~style.css — page styling
~notes.db — stored notes database
